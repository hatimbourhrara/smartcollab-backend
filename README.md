# SmartCollab Backend

SmartCollab is a modern collaboration and project management backend built with Java, Spring Boot, PostgreSQL, Docker, and JWT Authentication.

## Project Goal

The goal of this project is to build a professional backend system step by step and improve backend engineering skills.

## Tech Stack

- Java
- Spring Boot
- PostgreSQL
- Docker
- JWT Authentication

## Planned Features

- User authentication
- Teams
- Projects
- Tasks
- Comments
- Notifications
- Roles and permissions
- Activity tracking

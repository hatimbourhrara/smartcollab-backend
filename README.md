# SmartCollab Backend

SmartCollab is a modern collaboration and project management backend built with Java, Spring Boot, PostgreSQL, Docker, and REST APIs.

## Project Goal

The goal of SmartCollab is to build a professional backend system for a collaboration and project management platform, inspired by tools like Trello, Jira, Notion, and Discord.

This project is designed as a long-term software engineering project to practice backend architecture, clean code, databases, API design, authentication, cloud deployment, and modern engineering workflows.

## Tech Stack

- Java
- Spring Boot
- PostgreSQL
- Docker
- Docker Compose
- Spring Data JPA
- Hibernate
- REST APIs
- Maven

## Current Features

- User registration endpoint
- Task creation
- Get all tasks
- Get task by ID
- Update task
- Delete task
- Update task status
- Filter tasks by status
- PostgreSQL database integration
- Docker-based database setup
- Global exception handling
- Professional API error responses
